How do we quickly estimate average temperature on a planet?

Requirements:

  • Any location
  • Any point in time
  • Any timestep length
  • Any stable stellar system
  • In under 10ms

Deep Dive

Conclusions:

  • can't suffice with single equation, must work in general case
  • "on-rails" orbital mechanics, assume stability
  • precision may be an issue, must work from any reference frame ("Anti-Kraken" clause)
  • potentially ergodic, sample multiple positions when timestep is large and average between them
