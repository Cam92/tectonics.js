Aw, Come On!

Tectonics.js does not support this version of Internet Explorer

So here's some background:

Internet Explorer® ("IE") is a software utility packaged with the Microsoft® Windows® operating system. Microsoft® packages its products with Internet Explorer® as a service to users, who may then use Internet Explorer® to acquire an actual web browser.

All but the most very recent versions of Internet Explorer® are noncompliant with the international standards that are expected among modern web browsers. Tectonics.js uses state of the art web technologies and as such requires support for the latest web standards, sine qua non. Without a compliant browser the project is simply not possible, and even delivering simple messages such as what you see here requires work arounds on the developer's behalf.

It is strongly advised that you install a compliant web browser. The following are all excellent choices that are easily downloaded at no cost to you, whatsoever. Most browsers here offer autoupdate features that ensure your browser supports even the latest web standards with little or no effort on your part.

  • Google Chrome
  • Mozilla Firefox
  • Opera
  • Apple Safari

Alternatively, it is currently possible for users to run Tectonics.js on the latest version of Internet Explorer®. You can update your copy of Internet Explorer® here. Please note that this is not guaranteed to be a long term solution. Future support for IE will not be taken into consideration as the situation progresses and Tectonics.js comes to adopt newer web technologies.

This probably all comes as a big downer, but cheer up then: IE offers bar none the best support for marquee text in the whole world!
