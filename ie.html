Are you still using that version of IE?

Internet Explorer® is a software application packaged with the Microsoft® Windows® operating system. Microsoft® packages its products with Internet Explorer® as a service to users, who may then use the utility to acquire their own choice of web browser.

All but the most recent versions of Internet Explorer® are noncompliant with the international standards that the internet relies upon. Tectonics.js uses state of the art web technologies and as such requires support for the latest web standards, sine qua non. Without a compliant browser the project is simply not possible, and even delivering simple messages such as this one requires work arounds on the developer's behalf.

It is strongly advised that you install a compliant web browser. The following are all excellent choices that are easily downloaded at no cost to you, whatsoever. Most browsers here offer autoupdate features that ensure your browser supports even the latest web standards with little or no effort on your part.

  • Google Chrome
  • Mozilla Firefox
  • Opera
  • Apple Safari

Alternatively, it is currently possible for users to run Tectonics.js on the latest version of Internet Explorer®. You may be able to update your version of IE using your copy of the Official Microsoft® Windows® Update® Utility®. Please note that future support for the current version of IE will not be taken into consideration as the project comes to adopt newer web technologies.

This probably all comes as one big downer, but cheer up then. Internet explorer offers bar none the best support for scrolling text there is!
